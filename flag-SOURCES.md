# 历史旗帜素材来源

## 1933—1935年德国共同国旗

- 黑—白—红旗：以CSS三色条准确绘制。
- 纳粹党旗：`flag-germany-nsdap-1920-1945.svg`
- 来源：[Wikimedia Commons — Flag of the NSDAP (1920–1945)](https://commons.wikimedia.org/wiki/File:Flag_of_the_NSDAP_(1920%E2%80%931945).svg)
- 原始文件标注作者：German Government
- 许可：Public domain
- 用途：仅在1933年3月12日至1935年9月15日与黑—白—红旗并列显示，表达两面共同国旗；不用于其他国家或年份。


## CSS绘制的常见国旗图标

- 本批新增的英国、美国、意大利、西班牙、葡萄牙、北欧十字旗、巴尔干三色旗、奥斯曼/土耳其、苏联、中日印巴等旗帜均用 CSS 渐变绘制，不新增外部图片素材。
- 对小尺寸内难以准确表现的纹章、星月、白日、五星、锤镰、法轮等复杂元素，页面标签会注明“简化”或“小尺寸近似”，只作为地图名称旁的识别图标，不作为可复用的精确旗帜素材。
- 颜色和采用年份按常见历史旗制做阶段化处理；后续若引入 SVG 旗帜资产，需要在本文件继续补充逐项来源与许可。


## 本轮历史时期校正

- 奥匈帝国：奥匈双元君主国没有单一通用国旗，页面不再把奥地利帝国黑黄旗直接作为奥匈帝国国旗，而是以奥地利红白红与匈牙利红白绿并列的小图标表示。参考：Flags of Austria-Hungary 对“没有 commonly used universal flag”的说明：https://en.wikipedia.org/wiki/Flags_of_Austria-Hungary
- 西班牙：1931—1939 第二共和国阶段使用红—黄—紫三色旗；其他阶段继续使用红—黄—红简化旗。参考：https://en.wikipedia.org/wiki/Flag_of_the_Second_Spanish_Republic
- 葡萄牙：1910革命前显示王国蓝白旗简化图标，1910以后显示共和国绿红旗简化图标。参考：https://en.wikipedia.org/wiki/Flag_of_Portugal
- 法国：1815—1830波旁复辟阶段显示白旗简化图标，1794以后其他适用阶段使用三色旗。


## 本轮补充的殖民地与非欧洲大国旗帜

- 加拿大、澳大利亚、新西兰、英属印度等英帝国相关实体：使用带英国旗区的红/蓝船旗小尺寸简化图标，徽章与星徽不逐点绘制。加拿大红船旗参考加拿大政府“Royal Union Flag was used ... until 1965”及加拿大红船旗史料。
- 清朝：1889—1912黄龙旗在小尺寸中简化为黄色底与红珠提示，龙纹不逐点绘制。
- 波斯/伊朗：1907以后狮子与太阳旗简化为绿白红三色，中心纹章不逐点绘制。
- 巴西：1889共和国旗简化为绿底蓝圆，黄色菱形与星空不逐点绘制；此前帝国旗也以同一绿色体系做占位，标签标明纹章简化。
- 墨西哥、阿根廷、南非联邦、瑞士、卢森堡、爱尔兰、冰岛、波罗的海国家、阿富汗、埃及等新增旗帜均为 CSS 小图标，复杂纹章统一标注简化。
